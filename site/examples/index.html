    Examples¶

    Explore practical examples and sample applications demonstrating how to use GraphAI Utils to build powerful AI-driven solutions. Each example provides working code that you can adapt for your own projects.

    Featured Examples¶

    • Basic OpenAI Server

      A minimal Express.js server using @receptron/graphai_express to provide an OpenAI-compatible API endpoint.

      Focus: Core server setup, API compatibility.

      View Example

    • Vue Graph Dashboard

      An interactive dashboard built with Vue.js and @receptron/graphai_vue_cytoscape to visualize and manipulate GraphAI workflows.

      Focus: Frontend visualization, Vue integration, Cytoscape.

      View Example

    • React Flow Visualizer

      A React application showcasing graph visualization and workflow editing using @receptron/graphai_react_cytoscape.

      Focus: Frontend visualization, React integration, dynamic graph updates.

      View Example

    • Firebase Cloud Deployment

      A complete example of deploying a GraphAI application (backend function and frontend) to Firebase.

      Focus: Serverless deployment, Firebase Functions, Firebase Hosting.

      View Example

    • Custom Agents Showcase

      Examples of creating and using custom GraphAI agents for specialized tasks within a workflow.

      Focus: Agent development, extending GraphAI capabilities.

      View Example

    Example Categories¶

    Backend / API Examples¶

    • Basic OpenAI Server: Minimal setup for OpenAI API compatibility.
    • Streaming API Endpoint: Demonstrates setting up an endpoint for streaming responses (often part of basic-server.md or a dedicated example).
    • Authenticated API: Shows how to add API key authentication to your GraphAI server.

    Frontend / Visualization Examples¶

    • Vue Graph Dashboard: Comprehensive Vue.js integration.
    • React Flow Visualizer: Detailed React component usage.
    • Dynamic Graph Loading: Example of fetching graph data from an API and rendering it.

    Full-Stack Examples¶

    • Chat Application: A full-stack chat application using GraphAI for the backend and a Vue/React frontend. (Combines concepts from multiple examples).
    • Firebase Cloud Deployment: End-to-end deployment of a GraphAI app.

    Agent & Workflow Examples¶

    • Custom Agents Showcase: Focus on building unique agent functionalities.
    • Complex Workflow: An example demonstrating a multi-step, conditional GraphAI workflow.
    • Data Processing Pipeline: Using GraphAI for a data transformation or enrichment pipeline.

    How to Use These Examples¶

    1. Browse: Find an example that matches your interest or use case.
    2. Understand: Read the description and the key features highlighted.
    3. Code: Most examples will link to a GitHub directory or provide self-contained code snippets.
    4. Run: Follow the instructions to run the example locally.
    5. Adapt: Modify the code for your own project needs.

    Contributing Examples¶

    We welcome contributions! If you have built an interesting application or a useful pattern with GraphAI Utils, please consider sharing it.

    • Fork the repository.
    • Add your example in a new Markdown file within the docs/examples/ directory.
    • Provide clear instructions and code snippets.
    • Submit a pull request.

    See our contribution guidelines for more details.

    Need Help?¶

    If you have trouble running an example or have questions:

    • Check the Troubleshooting Guide.
    • Ask in our GitHub Discussions.
    • Report an issue on GitHub Issues if you find a bug in an example.
